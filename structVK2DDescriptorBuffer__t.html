Detailed Description

Automates memory management for uniform buffers and the lot.

The intended usage is as follows:

  1. Create the descriptor buffer
  2. Call vk2dDescriptorBufferBeginFrame before beginning to draw 3.
  3. Call vk2dDescriptorBufferEndFrame before submitting the queue

It will put a event barrier into the startframe command buffer where drawing happens, and then it will put a memory copy into the second command buffer at the end of the frame so Vulkan copies the buffer to device memory all at once instead of tiny increments.
