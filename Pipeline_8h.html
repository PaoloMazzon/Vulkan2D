Creates a graphics pipeline.

Parameters
devDevice to create the pipeline with
renderPassRender pass that will be used with the pipeline
widthWidth of the screen (used for scissor)
heightHeight of the screen (used for scissor)
vertBufferBuffer containing the compiled SPIR-V vertex buffer
vertSizeSize of the aforementioned buffer in bytes
fragBufferBuffer containing the compiled SPIR-V fragment buffer
fragSizeSize of the aforementioned buffer in bytes
setLayoutDescribes the shader's input information
vertexInfoTells the pipeline how to read input vertices
fillIf true, polygons are drawn as wireframe and not filled
msaaWhat level of MSAA to use
Returns
Returns the new pipeline or NULL if it failed

Several things are set to dynamic state and will be ignored, hence this asking for only 3 create infos when in reality there are many more. For a complete list of dynamic state,

  • Viewports
  • Rasterization line width (if wireframe is enabled)
  • Scissor

(This may change in the future) .
